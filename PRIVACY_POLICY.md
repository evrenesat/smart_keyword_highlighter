Privacy Policy — Keyword Highlighter

Data Collection:
This extension does not collect or transmit personal data or browsing data to any server. All text processing happens locally on your device to apply highlights.

Data Use:
The extension reads page content only to detect and highlight keywords as described in the product listing and options page.

Data Storage:
User preferences and a local keyword registry are stored in the browser’s local storage (storage.local). This data never leaves the device and can be cleared by the user at any time.

Data Sharing:
No data is shared with third parties. No analytics or advertising services are used.
